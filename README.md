Project Overview

This project is a real-time bag counting system built using YOLOv8 and OpenCV.
It detects and tracks bags in a video and counts them when they cross a defined line.
The system uses object tracking to avoid duplicate counting and works efficiently on recorded video input.
This project demonstrates practical implementation of:

Object Detection
Object Tracking
Computer Vision
Real-time Video Processing

Features

Detects bag-related objects using YOLOv8
Tracks objects with unique IDs
Counts bags when they cross a defined line
Prevents duplicate counting
Works on pre-recorded video
Lightweight and fast using YOLOv8 Nano

Technologies Used

Python
OpenCV
Ultralytics YOLOv8
PyTorch
